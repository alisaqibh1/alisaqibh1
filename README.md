**🩺 [SkinSense AI](https://skinsenseai-pk.vercel.app)** — Skin Disease Detection Platform  
Full-stack healthcare app detecting 31 skin diseases from images using a DINOv2-based deep learning model, with Groq (Llama 3) generating structured medical advice. React · Tailwind CSS · FastAPI · PyTorch · Groq API

**💬 Azure AI Foundry Chat App** — Conversational AI with Persistent Context  
ChatGPT-style chat application built with FastAPI and Azure AI Foundry, using the OpenAI Responses API with secure Azure Identity authentication and multi-turn conversation tracking. Python · Azure OpenAI · Azure Identity

**🖼️ [Multimodal Vision Chat](https://github.com/alisaqibh1/multimodal-vision-chat-azure)** — Vision-Enabled AI Assistant  
A multimodal AI chat application built with Microsoft Azure AI Foundry (GPT-5-mini), demonstrating vision-based use cases including recipe suggestions from food images and detailed image description/identification. Python · Azure AI Foundry · OpenAI Responses API · Azure Identity

**🎯 AI Career Path** — AI-Powered Career Guidance Platform  
Full-stack platform delivering personalized career recommendations via Google Gemini AI, automated study timetables via Groq API, and a dual-portal RBAC system for students and employers. React (Vite) · FastAPI · PostgreSQL · JWT

**📊 [ByteCRM](https://bytecrm.vercel.app)** — Customer Relationship Management Platform  
CRM platform with 60+ REST API endpoints, real-time analytics dashboard, and email campaign automation. React · Node.js · Express · MongoDB

**🏡 [LuxuryHomes](https://luxuryhomes-frontend.vercel.app)** — Real Estate Platform  
Full-stack luxury real estate platform with dynamic search/filtering, GSAP animations, and an admin analytics dashboard. React · Vite · Node.js · MongoDB

**📅 ByteSlot** — Scheduling & Booking Platform  
Responsive booking platform with event-type configuration and end-to-end appointment workflows. React · Vite · Tailwind CSS

**🚗 GearAid (Final Year Project)** — Vehicle Assistance App
